This is my personal OCR application, using react native technology, combining with expo for the app platform.

## Running backend

1. Install packages

    ```bash
    bun install
    ```

2. Start the server
    ```bash
    bun start
    ```

## Running frontend

1. Install dependencies

    ```bash
    npm install
    ```

2. Start the app

    ```bash
     npx expo start
    ```
